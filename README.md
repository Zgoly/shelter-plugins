# Shelter Plugins

A collection of plugins for [shelter](https://shelter.uwu.network/).

## Available Plugins

| Plugin | Link | Description |
|--------|------|-------------|
| `e2ee` | [`plugins/e2ee`](https://zgoly.github.io/shelter-plugins/plugins/e2ee) | End-to-end encryption for Discord messages with auto-decrypt and custom UI |

> [!NOTE]
> These plugins are written in **vanilla JavaScript** and do **not** use the [Lune](https://github.com/uwu/shelter-template) build toolchain. There is no compilation step, no JSX, and no bundler involved. Each plugin consists of raw `plugin.js` and `plugin.json` files hosted directly via GitHub Pages.
